# Underwater-Marine-Waste-And-Pollution-Detection-Using-Deep-Learning
AI-powered underwater pollution detection system using CNNs, Computer Vision, FastAPI, Docker, and MLflow for real-time marine waste classification and deployment.
Underwater Pollution Detection System

An AI-powered computer vision system designed to detect and classify marine waste from underwater images using deep learning. The project leverages transfer learning with Convolutional Neural Networks (CNNs) to identify different categories of underwater pollution while addressing challenges such as low visibility, varying lighting conditions, and image distortion.

The complete pipeline includes data preprocessing, model training, experiment tracking, model deployment, and REST API integration, making it suitable for real-world environmental monitoring and autonomous robotic perception systems.

Key Features
🌊 Detects and classifies multiple categories of marine waste
🤖 Transfer Learning using ResNet architecture
📷 Image preprocessing with contrast enhancement and histogram equalization
📈 Experiment tracking using MLflow
⚡ FastAPI-based REST API for real-time inference
🐳 Dockerized deployment for portability
🔄 ONNX export for cross-platform inference
📊 Performance evaluation using Precision, Recall, F1-Score, and mAP
📁 Modular and production-ready project structure
